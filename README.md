# Notes App

Aplicação de notas desenvolvida com JavaScript Vanilla.

O utilizador pode:

- criar notas
- editar notas
- apagar notas
- marcar favoritas
- pesquisar notas
- filtrar favoritas

As notas ficam guardadas no localStorage.

---

## Funcionalidades

- CRUD completo de notas
- Sistema de favoritos
- Pesquisa por título
- Filtros dinâmicos
- Modal de edição
- Persistência com localStorage
- Projeto modularizado

---

## Tecnologias

- HTML5
- CSS3
- JavaScript (ES Modules)
